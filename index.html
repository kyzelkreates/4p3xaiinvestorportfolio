Safer Logistics & Bridge Strike Risk Reduction

4P3X Verse™ route and fleet products can support safer logistics by helping drivers, controllers, and vehicle operators plan journeys with clearer awareness of vehicle height, width, weight, length, restrictions, hazards, route confidence, and legal-critical information.

The system is designed to improve awareness and support structured checks — not to guarantee a legally safe or incident-free route. Human responsibility, controller review, and driver judgement remain essential.

📋
Vehicle Profile Checks

Pre-departure structured check against the registered vehicle profile — confirming dimensions and category are logged before the journey begins.

📐
Height / Width / Weight / Length Awareness

Full four-dimension vehicle profile — height, width, gross weight, and vehicle length — matched against known route restrictions, low bridges, narrow roads, weight-limited crossings, and length-restricted junctions.

⚠️
Route Hazard Warnings

Flagged hazard points along a planned route — including low bridges, height-restricted roads, weight limits, width restrictions, and length-critical junctions — surfaced before departure, not during.

✅
Driver Acknowledgement Before Departure

A structured digital acknowledgement step — the driver confirms vehicle dimensions, reviews flagged hazards, and accepts responsibility before the journey is logged as cleared. Creates an auditable pre-departure record.

🖥️
Controller Dashboard Oversight

Controller-facing dashboard showing fleet vehicle profiles, pending acknowledgements, route confidence scores, flagged hazards, and journey status. Supports oversight without replacing controller judgement.

📊
Route Confidence Scoring

A calculated confidence indicator based on vehicle height, width, weight, and length matched against known route restriction data. Scored as High / Medium / Low confidence — giving drivers and controllers a structured signal, not a guarantee. Score drops when any dimension approaches or exceeds a known restriction threshold on the planned route.

High Confidence No flagged restrictions on planned route for this vehicle profile
Medium Confidence One or more restrictions flagged — review recommended before departure
Low Confidence Route has known restriction conflicts with this vehicle — alternative route or controller sign-off required

Route confidence scoring supports decision-making only. It does not replace route planning expertise, legal checks, or controller approval.

⚠ Security Note — 4P3X API Config Guard™

Live features require Vercel environment variables and serverless routes. Never place API keys in frontend code.

  • Never expose: TextBee keys, OpenAI/Groq keys, Stripe secrets, Supabase service role keys, database URLs, JWT secrets, webhook secrets, GitHub tokens
  • No secrets in: localStorage, public frontend files, README examples, screenshots, or exported ZIPs
  • Use /api/textbee-lead.js for all live notification routing — server-side credentials only
  • If credentials were accidentally exposed: revoke immediately and regenerate before redeployment
