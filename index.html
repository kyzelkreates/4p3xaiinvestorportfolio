⚠ Security Note — 4P3X API Config Guard™

Live features require Vercel environment variables and serverless routes. Never place API keys in frontend code.

  • Never expose: TextBee keys, OpenAI/Groq keys, Stripe secrets, Supabase service role keys, database URLs, JWT secrets, webhook secrets, GitHub tokens
  • No secrets in: localStorage, public frontend files, README examples, screenshots, or exported ZIPs
  • Use /api/textbee-lead.js for all live notification routing — server-side credentials only
  • If credentials were accidentally exposed: revoke immediately and regenerate before redeployment
